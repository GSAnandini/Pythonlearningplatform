Introduction

About python

* Python is a high-level programming language.

* It was invented by Guido-Van-Rossum and released in 2001.

* It is both procedural and object oriented.

* Python uses interpreter which converts the code written in python language by the user to the language which computer hardware or system can understand.

Keywords

Python has a set of keywords that are reserved words that cannot be used as variable names, function names, or any other identifiers

There total 33 keywords in python.

Identifiers

"Identifiers" or "symbols" are the names you supply for variables, types, functions, and labels in your program.

Rules of Identifiers

* Identifiers can be a combination of letters in lowercase (a to z) or uppercase (A to Z) or digits (0 to 9) or an underscore _ .

* An identifier cannot start with a digit.

* Keywords cannot be used as identifiers.

* We cannot use special symbols like !, @, #, $, % etc.

* An identifier can be of any length.
