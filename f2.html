About Variables

Variables

A variable is the name given to a memory location in a program.

Simply, Variable is a container to store a value.

For Example:

a = 30

Here,

a -> variable_name


Few more examples:

b = "virat"

c = 71.23


Rules for defining a variable:

* A variable name can contain alphabets, digits and underscores.

* A variable name can only start with an alphabet and underscore not with digits or special symbols.

* No white spaces are allowed to be used inside a variable name.
